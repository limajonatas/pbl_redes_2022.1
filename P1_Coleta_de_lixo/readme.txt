- O projeto está subdivido em 4 "subprojetos" que são eles: o projeto do servidor 
  (Nuvem IOT descentralizada), administrador, lixeira e caminhão, afim de poder ser 
  excutado separadamente em máquinas distintas.
  
- Uso das bibliotecas 'JSON' e 'FlatLaf' - ao abrir os projetos pode apresentar erros em 
  relação às bibliotecas. Neste caso é necessário adicionar as bibliotecas, elas estão 
  junto ao projeto. Em cada subprojeto há uma pasta chamada 'lib' onde está as duas 
  bibliotecas. Além disso, haverá na raiz do projeto 'P1_Coleta_de_lixo' uma pasta 
  chamada 'lib' também, com as bibliotecas.

- Execução: cada projeto é executado através de uma de suas classes que estão 
    dentro de um pacote nomeado pelo mesmo nome; assim também a classe para executar o
    projeto, que recebe o mesmo nome.
      EX: projeto 'Nuvem': pacote: 'nuvem', classe para executar o projeto 'Nuvem'.

- Projeto feito usando o Apache NetBeans 13 - jre 11 ou superior 
  (obs.: estava instalado na máquina que foi feito o projeto o java 14).
